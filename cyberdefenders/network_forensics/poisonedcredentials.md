## Scenario
The organization's security team has detected a surge in suspicious network activity.<br>
There are concerns that LLMNR (Link-Local Multicast Name Resolution) and NBT-NS (NetBIOS Name Service) poisoning attacks may be occurring within the network.<br>
These attacks are known for exploiting these protocols to intercept network traffic and potentially compromise user credentials.<br>
The task is to investigate the network logs and examine captured network traffic.
